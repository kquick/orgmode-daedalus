* orgmode-daedalus

   A parser and pretty-printer for the [[https://orgmode.org][Org-Mode]] document format.

   - Parsing is implemented using the Galois [[https://github.com/GaloisInc/daedalus][Daedalus]] parser (which permits
     generation of C, C++, and Rust parsers as well).

     + Parsing errors are reported via the [[https://hackage.haskell.org/package/diagnose][diagnose]] package.

   - Pretty-printing is implemented using the [[https://hackage.haskell.org/package/sayable][sayable]] package.

Development and usage is supported by the [[https://github.com/GaloisInc/eng][eng]] tool.  Install this tool and type
~eng~ in this directory for more information.

* Alternatives

** Parsing

   - [[https://hackage.haskell.org/package/org-mode][org-mode]]
     + list items can only be simple text
     + not responsive to github engagement
     + supports output as minimal org source, but no pretty-printing
   - [[https://hackage.haskell.org/package/org-parser][org-parser]]
   - [[https://hackage.haskell.org/package/orgmode][orgmode]]
     + built on Parsec library
   - [[https://hackage.haskell.org/package/orgmode-parse][orgmode-parse]]
     + built on Attoparsec
     + no markup or block support
     + no pretty-printing
